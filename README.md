# Giovanni Nicolazzo — Website

## Local preview
If the PR preview is not available, you can run a local preview from the repository root:

```bash
python -m http.server 8000
```

Then open:

- http://127.0.0.1:8000/
- http://127.0.0.1:8000/index.html

This repository currently contains the generated static site, so local preview does not require a build step.
